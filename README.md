# ad_block 浏览器广告拦截插件
使用可自定义的广告拦截器提升您的浏览速度，使您能够以自己的方式过滤内容。阻止侵入性广告、跟踪器和特定页面元素，同时将受信任的网站列入白名单。享受更快的加载速度、增强的隐私和跨设备的定制 Web 体验。
